# 🔧 ESP32 Predictive Maintenance System

A multi-sensor based predictive maintenance system built using ESP32.  
This project monitors machine conditions like temperature, vibration, load, and environment to detect potential failures.

---

## 🚀 Features
- Real-time sensor monitoring
- Multi-sensor data fusion
- Basic predictive logic (threshold + condition-based)
- Serial output with system status

---

## 🧩 Sensors Used
- DS18B20 (Temperature)
- MPU6050 (Vibration)
- Potentiometer (Load simulation)
- LDR (Light/Environment)

---

## ⚙️ Hardware
- ESP32
- Breadboard + jumper wires

---

## 🧠 Working Logic
- High temperature + high load + high vibration → Critical condition
- Moderate values → Warning
- Normal values → Safe

---
